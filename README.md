This repository contains all the source code for the practical exercises of the **Object-Oriented Programming** course at the University of Information Technology (UIT). All exercises are written in **C++**.

The repository is divided into folders corresponding to each practice week. Each folder includes Visual Studio Solution files and source code addressing problems related to various OOP concepts:

## Tools & Development Environment

* **Language:** C++
* **IDE:** Microsoft Visual Studio 2026
* **OS:** Windows

## How to Run the Project

1. Clone this repository to your local machine:
   ```bash
   git clone [https://github.com/tanphatk7/Object-Oriented-Programming.git](https://github.com/tanphatk7/Object-Oriented-Programming.git)
2. Open the corresponding practice folder (e.g., TH02).
3. Double-click the **.sln** file to open the Solution in Visual Studio.
4. Select **Build** -> **Build Solution** (or press **Ctrl + Shift + B**).
5. Run the program by pressing **F5** (or **Ctrl + F5**).
